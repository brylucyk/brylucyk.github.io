This "Major League Baseball: Data Retrieval & Modification Form" was created as a project for INFO 1234: Web Scripting. The form retrieves JSON data for a given date, using an asynchronous XMLHttpRequest. Users can cycle through the information for each game played on the specified date. I also customized the form to show when no data was available for a given date.
